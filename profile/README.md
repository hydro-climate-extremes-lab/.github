[H-CEL](https://www.ugent.be/bw/environment/en/research/h-cel) is a research lab at Ghent University studying global hydrology, climate, ecology & land-atmosphere interactions using earth observation datacubes, climate and hydrological models.

In this GitHub organisation, we host public code, documentation and teaching material. 

## Research

- Source code of the [Heat And MoiSture Tracking framEwoRk (HAMSTER)](https://doi.org/10.5194/gmd-15-1875-2022
) for the evaluation of Lagrangian models, both for the [original](https://github.com/h-cel/hamster) and [updated](https://github.com/h-cel/HAMSTER-v2) version. 

## Teaching

The lab is active in providing education on hydrology and climate at the [Faculty of Bioscience Engineering](https://www.ugent.be/bw/en/education). The list below consists of course material made with [Quarto](https://quarto.org/) for computer exercises:

- [Land-Atmosphere Interactions](https://github.com/h-cel/land-atmosphere-interactions-public): using [CLASS](https://classmodel.github.io/) to explore land-atmosphere feedbacks
- [Hydrological Modelling](https://github.com/h-cel/hydrological-modelling-public): rainfall-runoff modelling with a conceptual model
